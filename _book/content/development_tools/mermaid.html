Mermaid

Mermaid

Mermaid is simple markdown-like script language for generating charts from text via javascript 1

Syntax

TB/TD - top bottom
BT - bottom top
RL - right left
LR - left right

Single Node

Node

A node is single a box/entity

graph LR
    id1
graph LR id1

Node with Title

graph LR
    id1[This is the text in the box]
graph LR id1[This is the text in the box]

Node with Round Borders

graph LR
    id1(This is the text in the box)
graph LR id1(This is the text in the box)

Node with Circle

graph LR
    id1((This is the text in the circle))
graph LR id1((This is the text in the circle))

Node with Asymmetric Shape

graph LR
    id1>This is the text in the box]
graph LR id1>This is the text in the box]

Node with Rhombus

graph LR
    id1{This is the text in the box}
graph LR id1{This is the text in the box}

Links Between Nodes

Arrow Head

graph LR
  A-->B
graph LR A-->B 
graph LR
  A---B
graph LR A---B

Link with Text

graph LR
  A-- This is the text -->B
graph LR A-- This is the text -->B 
graph LR
  A-- This is the text ---B
graph LR A-- This is the text ---B

Dotted Link

graph LR
  A-.-B
graph LR A-.-B 
graph LR
  A-.->B
graph LR A-.->B 
graph LR
  A-. This is the text .-B
graph LR A-. This is the text .-B 
graph LR
  A-. This is the text .->B
graph LR A-. This is the text .->B

Thick Link

graph LR
  A==>B
graph LR A==>B 
graph LR
  A===B
graph LR A===B 
graph LR
  A== This is the text ===B
graph LR A== This is the text ===B 
graph LR
  A== This is the text ==>B
graph LR A== This is the text ==>B

Flowchart

Top Down

graph TD
  A-->B
  A-->C
  B-->D
  C-->D
graph TD A-->B A-->C B-->D C-->D

Left Right

graph LR
  A-->B
  A-->C
  B-->D
  C-->D
graph LR A-->B A-->C B-->D C-->D

Special Characters

graph LR
  A["A double quote:#quot;"] -->B["A dec char:#9829;"]
graph LR A["A double quote:#quot;"] -->B["A dec char:#9829;"]

Link

graph LR;
  A-->B;
  click A callback "Tooltip for a callback"
  click B "http://www.github.com" "This is a tooltip for a link"
graph LR; A-->B; click A callback "Tooltip for a callback" click B "http://www.github.com" "This is a tooltip for a link"

Sequence Diagram

Syntax

-> a solid line without arrow
--> a dotted line without arrow
->> a solid line with arrowhead
-->> a dotted line with arrowhead
-x a solid line with a cross at the end (async)
--x a dotted line with a cross at the end (async)

Generation

sequenceDiagram
  participant John
  participant Alice
  Alice->>John: Hello John, how are you?
  John-->>Alice: Great!
sequenceDiagram participant John participant Alice Alice->>John: Hello John, how are you? John-->>Alice: Great!
